# JD Invest

Investment Analysis & Trading Tools

## Overview

JD Invest is a repository for investment research, analysis, and automated trading strategies.

## Getting Started

```bash
# Clone the repository
git clone https://github.com/baryshnikovd/JD-Invest.git
cd JD-Invest

# Set up Python virtual environment
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt
```
